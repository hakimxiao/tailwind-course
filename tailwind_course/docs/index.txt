#_container : 
    Container adalah element yang digunakan untuk membatasi lebar kontent agar tidak memenuhi seluruh lebar layar. 
    Kelas container di tailwind CSS secara otomatis menyesuaikan lebar elemen berdasarkan breakpoint yanhg telah 
    ditentukan. 
    @_Fungsi : 
        *   membuat layout yang rapi dan terpusat
        *   menyesuaikan lebar elemen berdasarkan ukuran layar (responsife)
        *   mencegah kontent meluas hingga tepi layar
        @_Breakpoint : 
            sm  : 640px
            md  : 768px
            lg  : 1024px
            xl  : 1280px
            2xl : 1536px

#_Pseudo Class 
    Digunakan untuk mengubah sebuah tampilan element berdasarkan kondisi tertentu, seperti hover, focus, first class dll.

#_Responsif Desain : Breakpoint & Media Queries
    Breakpoint adalh titik batas ukuran layar dimana tata letak atau gaya elemen berubah agar sesuai dengan berbagai perangkat
    (mobile, tablet, desktop). Media Queries adalah aturan CSS yang digunakan untuk menerapkan gaya berdasarkan ukuran layar.

    Tailwind menyediakan sistem breakpoint berbasis mobile-first, yang berarti semua gaya default berlaku untuk layar kecil, dan 
    kita dapat menimpa gaya tersebut untuk yang lebih besar menggunakan breakpoint classes.

#_Columns 
    Column adalah cara sederhana untuk membagu kontentdalam suatu elemen menjadi beberapa kolom. Biasanya digunakan untuk tata letak teks atau elemen yang mengalir seperti majalah.

    Tailwind menyediakan utilitas untuk bekerja dengan kolom menggunakan properti CSS seperti columns, yang mendukung berbagai ukuran dan tata 
    letak kolom. Tidak direkomendasikan pada hal kompleks, disini kolumn direkomendasikan untuk hal mudah seperti majalah 
    dan untuk yang kompleks kita bisa flex / grid.

#_FlexBox
    FlexBox(Flexible Box Layout) adalah model tata letak yang dirancang untuk mengatur dan menyelaraskan elemen dalam satu dimensi, baik itu secara 
    horizontal(baris) ataupun vertical(kolom). Flexbox sangat efektif untuk membuat tata letak yang fleksibel dan responsif,
    terutama saat elemen memiliki ukuran dinamis atau tidak diketahui sebelumnya.

    Flexbox mendukung berbagai properti untuk mengontrol distribusi ruang, penyelarasan, dan urutan elemen dalam suatu kontainer.
